# house-management-php-mysql
Full-stack house management system built with PHP and MySQL with login, CRUD operations, and relational database integration.

## Overview
This project is a PHP/MySQL web application for managing houses, owners, and user accounts through an authenticated dashboard.

## Features
- User login
- Session handling
- Owner management
- House management
- Password change
- Relational database integration

## Technologies
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Learning Outcomes
- Full-stack CRUD development
- Authentication and sessions
- Relational database design
- SQL joins and foreign keys
